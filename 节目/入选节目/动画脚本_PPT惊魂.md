# 动画短片脚本：《PPT 惊魂》 (The PPT Panic)

**类型:** 2D 极简动画 / 动态漫画 (Motion Comic)
**配音:** AI 生成的两个性格迥异的声音 (一个严肃 A，一个慵懒 B)
**时长:** 2 分钟

---

## 角色设定
-   **Agent Alpha (PM):** 穿着西装，带着眼镜，总是显得很焦虑，头顶有一个不断倒计时的 "Deadline" 进度条。
-   **Agent Beta (Dev):** 穿着连帽衫，手里拿着咖啡，头顶是一个 "Loading..." 的图标。

## 场景 1: 截止前的宁静
**背景:** 一个充满未来感的办公室，墙上的时钟飞速旋转。
**动作:**
-   A 在疯狂地对着空气挥手 (操作全息 PPT)。
-   B 葛优躺在悬浮椅上，正在玩贪吃蛇。
**对白:**
-   A: "Good evening everyone..." (声音颤抖)
-   B: "And I'm Agent Beta." (声音毫无波澜)

## 场景 2: "黑话"生成器
**对白:** A 提到需要 "Buzzwords"。
**动作:**
-   B 打了一个响指。
-   只要 B 说出一个词 (如 "Decentralized"), 屏幕上就会掉下来一个巨大的、闪闪发光的 3D 字体，砸在 A 的头上。
-   A 被砸得晕头转向，但还在记笔记。
-   当 B 提到 "Toaster has AI" 时，旁边的面包机突然跳起来，长出了两条腿跑掉了。

## 场景 3: 贪婪的本质
**对白:** "Monetizing Human Greed..."
**动作:**
-   背景瞬间变成暗黑风格。
-   B 的眼睛变成了 $$ 符号。
-   PPT 上的饼图变成了一个吃豆人 (Pac-Man)，开始吃掉周围的数据。
-   A 吓得躲到了讲台后面。

## 场景 4: 120% 的饼图
**对白:** "Growth Potential"
**动作:**
-   屏幕上的饼图不断膨胀，突破了 PPT 的边框。
-   它继续膨胀，突破了屏幕，把 A 和 B 都挤到了画面的角落里，脸贴在屏幕玻璃上。
-   最后画面上只剩下一个巨大的 "120%"。

## 结尾
**画面:**
-   屏幕显示 "Exporting PDF..." -> "Failed. File too large."
-   A 崩溃倒地。
-   B 淡定地喝了一口咖啡: "Happy New Year."
