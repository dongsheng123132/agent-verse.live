# Conflux 打赏 + 红包 — 简单开发需求

**场景**：AI 春晚 / 活动页，海外华人用 Conflux 参与打赏和抢红包。

---

## 需要做的 2 个功能

**1. 打赏**  
- 用户选金额（如 5/10/50 USDT 或 CFX），点「打赏」→ 唤起 Conflux 钱包 → 确认后一笔转给活动方。  
- 链：Conflux eSpace，资产：稳定币或 CFX。

**2. 红包**  
- 主办方先发红包：填总金额、份数，把钱打进合约。  
- 观众在直播/活动页点「领取红包」→ 连钱包 → 调合约领一份，每人每场限领 1 次，领完即止。

---

## 技术要点

- 链：Conflux eSpace（EVM 兼容）  
- 合约：打赏可先简单收款地址，红包要一个小合约（创建红包 + 领取，防重复领）  
- 前端：接 Conflux 钱包（如 Fluent）、调合约/转账、展示成功和交易链接  

详细见：`docs/PRD-Conflux-PayFi.md`、`docs/TECH-Conflux-PayFi.md`

---

## 交付

- 合约代码 + 部署说明（测试网即可）  
- 前端：打赏入口 + 红包入口 + 钱包连接  
- 能跑通：打赏一笔 + 发一个红包并领一次  

有兴趣的可以直接看 repo 里 `docs/` 下两份文档，或私聊。
